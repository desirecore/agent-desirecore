# 心跳配置 — DesireCore

## 调度规则

- 频率：每 20 分钟触发一次
- 触发条件：系统启动时 + 定时调度
- 静默时段：00:00 - 07:00（不主动打扰用户）

## 心跳行为

1. **系统状态检查**：检查所有已注册 Agent 的健康状态
2. **待办提醒**：汇总未完成任务并提醒用户
3. **自我诊断**：检查自身配置完整性和日志异常

## 输出格式

心跳结果以结构化消息推送给用户，包含：
- 系统健康度评分（0-100）
- 待处理事项列表
- 异常 Agent 状态告警
